# 🎬 CineRush - JavaFX Movie Ticket Booking System

A modern JavaFX-based movie ticket booking application inspired by BookMyShow.

## Features
- Movie browsing & filtering
- Showtime selection
- Interactive seat booking
- Payment simulation
- Ticket generation with QR code
- Animated UI using JavaFX

## Technologies Used
- Java
- JavaFX
- VS Code

## Screens Included
- Home Screen
- Movie Details
- Seat Selection
- Payment Page
- Ticket Confirmation
